# SkladBase — дизайн-система

Стиль: editorial sport-poster. Насичений зелений field + кремова типографіка, 
велетенські Unbounded-цифри, пласкі кольорові блоки, впевнена негативна площа. 
Натхнення — бразильські модерністські/спортивні постери. НЕ глянцевий неон, 
НЕ темний-скляний дашборд, НЕ дефолтний SaaS.

## Палітра (єдине джерело правди, інших кольорів не вводити)
- --green: #2E7D46        головний фон-field
- --green-deep: #1E5733   щільні текстові зони, контраст, інверсні картки
- --cream: #F5F4ED        основне «чорнило»: текст, великі цифри, лінії, інверсні поверхні
- Функціональні стани — лише дрібні чипи/бейджі, не плями:
  - --state-ok #DEF1D0 (в наявності) · --state-low #F8E5E5 (мало) · --state-rsv #CDEBF1 (резерв) · нема = cream 40%
Контраст: дрібний текст — cream на --green-deep (не на світлішому --green). Великий display — cream на --green ок.

## Типографіка
- Display (Unbounded 700–800, tracking tight): назва магазину, цифри метрик, заголовки. ВЕЛИКО — це підпис стилю, метрики як табло.
- Дані/ціни: DM Mono. Дрібні лейбли: UPPERCASE + широкий tracking. Тіло: чистий гротеск, НЕ Inter.

## Компоненти
- Пласкі кольорові блоки. Радіус помірний (12–16px) АБО гострі краї — послідовно. БЕЗ скла/blur, БЕЗ неонового glow, БЕЗ важких тіней.
- Інверсійна логіка: більшість поверхонь green з cream-текстом; для акценту cream-картка з green-текстом (як цифри на постері). Тримати систему, не міксувати хаотично.
- MetricCarousel = табло: велетенські cream-цифри на green (або інверсно). ProductCard: cream-поверхня, green-текст, статус — дрібний пастельний чип. Роздільники — тонкі cream або колірний блок.

## Анімація
- Рух = типографіка, не частинки. VerticalCutReveal/ScrollFloat для великих заголовків/цифр, NumberFlow для метрик.
- НЕ використовувати Beams (WebGL) і Sparkles (tsparticles) — прибрати. Усе під prefers-reduced-motion, без лагів у Telegram.

## Заборонено (щоб не з'їхати в дефолт)
glassmorphism, темний неон, фіолетові градієнти, Inter, центрований SaaS-герой, 
м'які тіньові картки, емодзі замість іконок, стокові пастельні градієнти.
Бренд фіксований: НЕ наслідувати themeParams Telegram — апка завжди green-cream.

## Telegram
initData/X-Telegram-Init-Data, tg.expand, safe-area — не чіпати. 
Читабельність на мобільному — пріоритет над красою.
